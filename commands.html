Git Commands

Overview

Git commands are instructions you type into a terminal or command-line interface to interact with Git, the version control system. These commands let you track, manage, and share changes in your project's files over time.

Generally, commands are used to...

  • Initialize a Git repository (git init)
  • Track changes to files (git add)
  • Save versions of your project (git commit)
  • Upload changes to a remote repository like GitHub (git push)
  • Download updates from a remote repository (git pull)
  • Check the status of your working directory (git status)
  • Navigate through commit history (git log)
  • Manage branches (git branch, git checkout, git merge)
