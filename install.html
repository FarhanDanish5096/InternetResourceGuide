How to Setup Git on Your Computer

Overview

Git, like many applications, require an intricate installation and setup before it can be utilized on your computer. There are options to bypass installation if you do not wish to use Git locally on your computer by exchanging information directly with GitHub.

GitHub, on the other hand, is an online webpage that can be accessed by any user from any location at any time. Therefore, no setup is required to access GitHub!

What are the steps to setup Git?

  1. Open the Git Downloads Page by accessing the hyperlink https://git-scm.com/downloads
  2. Click 'Download for Windows'.
  3. Click 'Click here to download
  4. Open the downloaded file from your Downloads tab
  5. Once opened, feel free to customize the configuration as you like or simply leave everything as default
  6. Congratulations! The installation is now complete and Git is ready to be used with your computer!
